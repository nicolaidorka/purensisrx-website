Purensis combines cutting-edge hardware and AI-powered software into a seamless automated workflow

Complete System Integration

Our platform unifies multiple advanced technologies to enable precise, automated tissue isolation and analysis. From imaging to collection, every step is optimized for accuracy and reproducibility.

Advanced Imaging System

Inverted Microscope System

Our dual-camera imaging system provides exceptional image quality with intelligent focus optimization:

  • Multi-tile scanning: Automated panoramic image acquisition across large tissue areas
  • Z-stack acquisition: Multiple focal planes captured for each tile
  • Intelligent focus selection: Scharr variance-based algorithm automatically identifies optimal focus plane
  • Multi-channel support: Seamless switching between brightfield, fluorescence, and other imaging modalities
  • High-precision stitching: Advanced algorithms create seamless panoramic images with coordinate metadata
Resolution: Up to 2048×1536 per tile
Pixel Precision: Dynamic calculation based on objective (e.g., 0.645 µm @ 10×)
Scan Area: Unlimited (tile-based)

AI-Powered Segmentation

AI Segmentation & Clustering

Leverage state-of-the-art deep learning for automated cell and region identification:

  • Deep learning segmentation: Pre-trained models for various cell types with customizable parameters
  • Interactive tuning: Real-time preview of segmentation results with adjustable thresholds
  • Clustering analysis: DBSCAN and K-Means algorithms group similar cells or regions
  • Polygon modification: Automated dilation, smoothing, and simplification tools
  • Manual refinement: Draw, edit, and merge annotations with precision measurement tools

Smart Analysis: Our unified analysis interface combines segmentation, clustering, and polygon modification in one streamlined workflow.

Precision Laser Microdissection

Automated Coordinate-Guided Cutting

Our laser system delivers sub-micrometer precision with intelligent automation:

  • Coordinate transformation: Automatic conversion from pixel annotations to stage coordinates
  • Laser offset calibration: Alignment workflow ensures cutting accuracy
  • Variable power control: Adjustable laser power for different tissue types (perimeter: 20%, push: 10%)
  • Leadscrew positioning: Optimal Z-axis positioning during initialization
  • Path optimization: Efficient cutting patterns minimize tissue damage
Cutting Speed: 2.0 mm/s (configurable)
Precision: Sub-micrometer accuracy
Power Range: 1-100% (tissue-adaptive)

Integrated Fluidics System

Automated Tissue Transport & Collection

Seamless integration of pumps and liquid handling for high-throughput sample processing:

  • Pump coordination: Synchronized inlet and outlet pumps for smooth tissue transport
  • Flow cell integration: Tissue flows through controlled microfluidic environment
  • Automated liquid handling: Automated collection into 96-well plates with well-specific targeting
  • Priming workflow: Automated system preparation ensures reliability
  • Safety features: Waste positioning and depth optimization prevent sample loss

High Throughput: Process and collect hundreds of tissue regions automatically, with each sample precisely tracked and stored.
